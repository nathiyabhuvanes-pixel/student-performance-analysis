Student Performance Analysis Project
1. Overview
This project focuses on analyzing the academic performance of 250 students using data analysis and machine learning techniques. The goal is to identify patterns, understand factors affecting performance, and build predictive models to estimate student outcomes. The analysis helps educators make better decisions to improve student success.

2. Dataset
The dataset contains records of 250 students with the following attributes:

Student ID
Gender
Age
Study Hours
Attendance (%)
Previous Exam Scores
Assignment Scores
Participation Level
Final Exam Score
Grade (Pass/Fail)

3. Objectives
To analyze student performance trends
To identify key factors influencing academic success
To visualize patterns in student data
To build a machine learning model for prediction
To improve academic decision-making using data
